# Premium Cybersecurity Portfolio & SOC Dashboard

A state-of-the-art, high-fidelity personal portfolio and Security Operations Center (SOC) dashboard. Engineered for performance, SEO, accessibility, and interactive visual aesthetics.

Built completely from scratch using **HTML5, CSS3, and modern Vanilla JavaScript (ES6+)** with zero build tools or packages, designed for direct deployment to GitHub Pages.

## ‚ö° Key Features

1.  **Interactive Terminal Shell:** A complete browser terminal running cyber commands (`help`, `about`, `projects`, `skills`, `experience`, `resume`, `contact`, `whoami`, `sudo`, `coffee`, `easteregg`) with typewriter animations.
2.  **SOC Decryption Loader:** A premium bootloader screen representing memory mapping and security system initialization sequences.
3.  **Active Cyber Lab Topology:** Interactive system architecture showing host configurations (Splunk, Windows/Linux VM targets) and MITRE ATT&CK technique tracking.
4.  **Premium Animation Layer:** Smooth animations and spotlights implemented using GSAP, custom Canvas particles, and Vanilla JS hover hooks.
5.  **PWA Support:** Installable on mobile and desktop, working offline via service workers.
